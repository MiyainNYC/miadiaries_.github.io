Some Cool Projects I Work On

My past projects include applying probabilistic models to solve various graph mining problems and conducting exhaustive exloratory analysis for business optimization. Here below are those I find particularly interesting:

Estimating the performance of employees is an important consideration in all organizations. With Software application development turning into an enormously profitable business, this project attempts to answer questions including identification of developers’ role, contribution, motivation, functionality and working preference, as well as in a broader sense, project membership, corporate cohesion, collaboration patterns and software development productivity. We demonstrate how network data science can be used to derive a broad spectrum of insights.
Recommender systems have been created for various data science applications in a variety of areas. We propose a neighborhood-based recommender system built on a graph database, an approach that managers could consult to assign employees to projects and thus help organizations achieve an improved allocation of human capital across multiple projects. We additionally demonstrate the simplicity and scalability of a graph-based collaborative filtering approach. It doesn’t necessarily require user features or large volume of data for training. Moreover, it helps solve the data sparsity and computation problem of memory-based approaches.
Mobile applications have turned into an enormously profitable business, with revenue expected to exceed 188 billion USD by 2020. These profits however are not distributed equally amongst developers, with forty-seven percent of developers making less than one-hundred USD, more than half of which make nothing at all. Creating a successful application is not easy。 In this project, we propose an ensemble model which combines popular ML algorithms such as Naive Bayes, Extra Trees and Gaussian Process, to predict apps success potential through analyzing text info (description, customer reviews, etc.) of thousands of Android game apps.
With a large dynamic network of NYC green cab business, this project delives deep-dive industry overview and insights to optimize cab drivers' profitability through an exhaustive exploratory analyis including time series analysis and graph analysis. We additionally propose a model combining random walk and pagerank to help drivers discover time-sensitive pickup opportunities.
